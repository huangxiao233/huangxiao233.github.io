#建立一个个人博客的方法

1.cd ~/Desktop 进入安全目录
2.github上面建立一个空的仓库，仓库名称是[你的用户名.github.io]
3.npm install -g hexo-cli 安装hexo
4.hexo new 文章名称 建立新文章，同时记录下路径
5.start 刚才记录下的路径.md 编辑这个md文件
6.start _config.yml 编辑网站配置
7.更改title,author,type: git
8.在最后一行新增一行，左边与type平齐，repo: 仓库地址
9.npm install hexo-deployer-git –save 安装git部署插件
10.hexo deploy hexo展开
